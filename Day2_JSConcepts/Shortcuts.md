## Tips

- Use `Ctrl + K Ctrl + S` to open the full keyboard shortcuts reference and search for any command.
- Use `Ctrl + /` to quickly comment or uncomment a line while coding JavaScript.
- Use `Ctrl + Shift + P` then type `>Preferences: Open Keyboard Shortcuts` to customize or search shortcuts.
