## Quick Code Examples

### Variables
```javascript
// Variable declaration with different keywords
var name = "John";        // Function-scoped, hoisted, can be re-declared
let age = 25;            // Block-scoped, hoisted (TDZ), cannot be re-declared
const salary = 50000;    // Block-scoped, cannot be reassigned

// Mutable vs Immutable
let mutable = 10;
mutable = 20;  // Can be changed

const immutable = 30;
immutable = 40;  // Error: Assignment to constant variable
```

### Operators
```javascript
// Arithmetic
let sum = 10 + 5;         // 15
let product = 10 * 2;     // 20
let power = 2 ** 3;       // 8

// Comparison
let isEqual = 10 === 10;  // true (strict equality)
let isLoose = 10 == "10"; // true (loose equality)

// Logical
let result = true && false;  // false
let either = true || false;  // true

// Ternary
let grade = score >= 80 ? "Pass" : "Fail";
```

### Conditional Statements
```javascript
// if-else
if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

// switch
switch (day) {
    case 1:
        console.log("Monday");
        break;
    default:
        console.log("Invalid day");
}
```

### Loops
```javascript
// for loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// while loop
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

// for...of (arrays)
const fruits = ["apple", "banana", "cherry"];
for (const fruit of fruits) {
    console.log(fruit);
}

// forEach method
fruits.forEach((fruit, index) => {
    console.log(index, fruit);
});
```

### Arrays
```javascript
// Creating arrays
const arr1 = [1, 2, 3];
const arr2 = new Array(1, 2, 3);
const arr3 = Array.of(1, 2, 3);
const arr4 = Array.from("hello");  // ['h', 'e', 'l', 'l', 'o']

// Array methods
arr.push(4);        // Add to end
arr.pop();          // Remove from end
arr.shift();        // Remove from start
arr.unshift(0);     // Add to start

// Search
arr.indexOf(2);     // Find index
arr.includes(2);    // Check existence
arr.find(x => x > 2);  // Find element

// Transform
arr.map(x => x * 2);     // Map values
arr.filter(x => x > 2);  // Filter elements
arr.reduce((sum, x) => sum + x, 0);  // Reduce to single value

// Sort
arr.sort((a, b) => a - b);  // Ascending order
```

### Functions
```javascript
// Function declaration
function add(a, b) {
    return a + b;
}

// Function expression
const multiply = function(a, b) {
    return a * b;
};

// Arrow function
const divide = (a, b) => a / b;

// Arrow function with multiple statements
const calculate = (a, b) => {
    const result = a + b;
    return result * 2;
};
```

## Key Takeaways

### Variable Declaration
- **var**: Function-scoped, hoisted to top, can be re-declared
- **let**: Block-scoped, hoisted but in Temporal Dead Zone, cannot be re-declared
- **const**: Block-scoped, hoisted but in TDZ, cannot be reassigned

### Equality Comparison
- **== (loose equality)**: Performs type coercion
- **=== (strict equality)**: No type coercion, preferred in JavaScript

### Array Methods Reference

| Method | Purpose | Returns |
|---|---|---|
| `push()` / `pop()` | Add/remove from end | Element / Array length |
| `shift()` / `unshift()` | Remove/add from start | Element / Array length |
| `map()` | Transform elements | New array |
| `filter()` | Select matching elements | New array |
| `reduce()` | Combine to single value | Single value |
| `find()` / `findIndex()` | Locate element | Element / Index |
| `sort()` | Order elements | Sorted array |
| `slice()` | Extract portion | New array |
| `concat()` | Join arrays | New array |

## Key Concepts Explained

### Variables (var, let, const)

```javascript
var x = 10;           // Function-scoped, can be re-declared
let y = 20;           // Block-scoped, cannot be re-declared
const z = 30;         // Block-scoped, immutable
```

### Identifiers Rules

- Must start with letter (a-z, A-Z), underscore (_), or dollar sign ($)
- Can contain digits (0-9) after first character
- Cannot contain spaces
- Case-sensitive
- Cannot start with digits
- Cannot be reserved keywords

### Literals

Fixed values directly written in code:

```javascript
var string = "Hello";     // String literal
var number = 42;          // Numeric literal
var boolean = true;       // Boolean literal
var empty = null;         // Null literal
var notAssigned;          // Undefined literal
```

### Operators

```javascript
// Arithmetic
let sum = 10 + 5;         // 15

// Comparison
let isEqual = 10 == 10;   // true

// Logical
let result = true && false; // false

// Assignment
let x = 5;
x += 3;                   // x = 8
```
